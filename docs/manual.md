# 1.予約フォームの使い方（お客様向け）

## 1.0 予約サイトへアクセス
本番URL: https://hyggely-reservation.web.app/

## 1.1 商品選択
1. 予約サイトにアクセス
2. 表示されている商品から予約したいものを選択
3. 「カートに追加」ボタンで商品を追加
4. 数量は「+」「-」ボタンで調整可能
5. 商品を選んだら「受取日時を選択」ボタンを押す

## 1.2 受取日時の選択
1. カレンダーから受取希望日を選択
   - 水曜日と土曜日のみ選択可能（営業日）
2. 受取時間帯を選択（11:00～17:00の1時間刻み）
3. 「お客様情報へ」ボタンを押す

## 1.3 お客様情報の入力
1. 以下の情報を入力
   - お名前（必須）
   - メールアドレス（必須）
   - 電話番号（必須・ハイフンなし）
   - 備考（任意）
2. 「確認画面へ」ボタンを押す

## 1.4 予約内容の確認
1. 注文内容、受取日時、お客様情報を確認
2. 問題なければ「予約を確定する」ボタンを押す
3. 修正したい場合は「戻る」ボタンで前の画面へ

## 1.5 予約完了
- 予約完了画面が表示されます
- 確認メールが登録したメールアドレスに届きます
- 受取当日は確認メールをお持ちください

# 2. 管理ダッシュボードの使い方（管理者向け）

## 2.1 ログイン
1. 管理画面URL: https://hyggely-reservation.web.app/admin/login にアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンを押す

## 2.2 ダッシュボード
ログイン後、以下の情報が表示されます：
- 本日の予約数
- 本日の売上
- 今月の売上
- 総顧客数
- 本日の予約一覧
- 在庫アラート（在庫が少ない商品）

## 2.3 予約管理
左メニューの「予約管理」をクリック
- 予約一覧の確認
- 予約ステータスの変更
  - 未確認 → 確認済み → 受取完了
  - キャンセル処理
- 予約詳細の確認

## 2.4 顧客管理
左メニューの「顧客管理」をクリック
- 顧客一覧の確認
- 顧客の予約履歴
- 顧客情報の編集

## 2.5 商品管理
左メニューの「商品管理」をクリック
- 商品の追加・編集・削除
- 在庫数の更新
- 商品画像のアップロード
- 販売可否の切り替え

## 2.6 店舗設定
左メニューの「店舗設定」をクリック
- 店舗情報の編集
- 営業日カレンダーの設定
- 臨時休業・臨時営業の設定
- メールテンプレートの編集

## 2.7 ログアウト
サイドバー下部の「ログアウト」ボタンをクリック

# 3. トラブルシューティング

## よくある問題

### Q: 予約ができない
A: 以下を確認してください
- 在庫があるか
- 営業日を選択しているか
- 必須項目がすべて入力されているか

### Q: ログインできない
A: 以下を確認してください
- メールアドレスが正しいか
- パスワードが正しいか
- 管理者として登録されているか

### Q: メールが届かない
A: 以下を確認してください
- メールアドレスが正しいか
- 迷惑メールフォルダを確認
- Firebase Functionsのログを確認
